## PHP语法
### PHP标记
1. ASP标记:<% PHP代码 %>  废弃
2. 短标记:<? PHP代码 ?>   废弃
3. 脚本标记:<script language="php"> php 代码 </script>  用得也少了
4. 标准标记:<?php php代码 ?>

### php注释
1. 行注释: // 注释内容
2. 行注释: # 注释内容
3. 块注释: /****  注释内容*/

### php语句分隔符
1. php中；作为分隔符
2. PHP中很多代码不是嵌入到html中，而是单身在一个文件中

### php变量
1. 变量是用来存储数据的
2. 变量是存在名字的
3. 变量是可以改变的
4. 变量是通过名字来访问数据的

### php变量的使用
1. 定义
2. 赋值：可以在定义的过程中赋值
3. 可以删除
4. php中不需要关键字
5. 所有的变量都需要使用 $符号

### php预定义变量
1. 预定义变量都是数组
   - $_GET: 获取所有表单以get方式提交的数据
   - $_POST:
   - $_REQUEST:
   - $_ENV：环境信息

### php可变变量
   - $a=b; $b=2; --> $$a=2

### php变量传值
1. 值传递
2. 引用传递 $a=1; $b=&$a;
3. 内存：栈区、代码段、数据段（全局区和静态区）、堆区
4. 过程：
  1. 代码装载
  2. 代码执行
  3. 脚本执行结束:回收所有内存

### php常量 const
1. define('PI',3.14); define('-_-','smile');
2. const PI = 3.14;
3. 常量命名规则 常量不需要使用$符，字母数字和下划线组成，不使用小写
4. 定义的时候必须赋值
5. 直接通过名字访问，不需要加$符号
6. 特殊符号使用constant(常量名)
7. 不允许被修改的数据，用常量

### php系统常量
1. 常用的系统常量
  - PHP_VERSION: PHP版本
  - PHP_INT_SIZE: 整型大小
  - PHP_INT_MAX: 整型能表示的最大值

### php文件包含
1. include/include_once
2. require
3. 被包含文件单独编译
4. include和require的区别
   - 包含不到文件的时候，报错的形式不一样
   - include级别比较轻，require出错后面不执行

### 文件加载路径
1. 绝对路径
   - 磁盘根目录开始
   - 网站根目录开始
2. 相对路径
   - ./
   - ../
3. 区别：绝对路径效率相对较低，安全;相对路径，效率高，容易出错

### 文件嵌套包含
1. 一个文件包含另外一个文件，同时被包含的文件又包含了另外一个文件
2. ./和 ../路径会受到嵌套包含影响


### 函数
1. function 关键字
2. 函数名
3. 参数：形参和实参
   - 允许实参多余形参
   - 参数不限制个数
   - 实参不能少于形参个数
4. 函数体
5. 返回值
6. 使用
7. eg:
  ```
  function display() {
      echo "hello world"
  }
  ```
8. 定义可以放在使用后面
9. 命名规范：字母数字和下划线组成，不能以数字开头
10. 函数一般规则:
    - 驼峰法：处理第一个单词，后面的每个单词首字母大写
    - 下划线法；单词之间下划线连接，其他都是小写
    - 一个脚本周期中不允许出现重名函数
11. 参数引用传递

### 作用域
1. 变量或常量能够被访问的区域
2. 分类
   - 全局变量:脚本内部
   - 局部变量:函数内部
   - 超全局变量:$_POST,$_SERVER 没有访问限制

### 静态变量
1. 在函数内部定义的变量
2. static 修饰

### 可变函数
1. 当前有一个变量所保存的值，刚好是一个函数的名字，那么可以使用变量+()来充当函数调用
2. 设置回调函数

### 匿名函数
1. 没有名字的函数 $func = function(){};
2. 本质拿到一个对象

### 闭包
1. 要执行的代码块
2. 函数内部有一些局部变量在函数执行完后未释放
   - 使用匿名函数
   - 匿名函数使用局部变量
   - 匿名函数返回外部使用
```
  function display() {
      $name = __FUNCTION__;
      $ func = function() use($name) {
          echo $name;
      }
      //$func();
      return $func;
  }
  $f = display();
  $f();
```
